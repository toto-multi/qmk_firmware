# 虎徹（こてつ）

**te96**と**Keyball**と**Gravity Keycaps**を組み合わせた47%分割キーボードです。

## 特徴

### ハードウェア

- 片手４段5列（小指外側なし）
- カラムスタッカード
- 狭ピッチ
- ロープロファイル（Choc Sunset tactile）
- 左手親指トラックボール
- バックライトLED
